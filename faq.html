Документация для администраторов

Администраторы сервера отвечают за обеспечение порядка и соблюдение правил среди игроков. В их обязанности входит:

  • Мониторинг чата и взаимодействие с игроками для предотвращения нарушений правил.
  • Рассмотрение жалоб от игроков и выявление конфликтных ситуаций.
  • Принятие мер в случае нарушений правил, включая выдачу варнов и банов.
  • Управление игровыми ресурсами и обеспечение их стабильной работы.
  • Сотрудничество с другими членами команды для разработки новых ивентов и улучшения игрового процесса.

Команды для администраторов

  • /kick [игрок] [причина] - выгнать игрока с сервера.
  • /ban [игрок] [время] [причина] - забанить игрока.
  • /pardon [игрок] - разбанить игрока.
  • /mute [игрок] [время] [причина] - замутить игрока.
  • /unmute [игрок] - размутить игрока.
  • /warn [игрок] [причина] - выдать варн игроку.
  • /tp [игрок1] [игрок2] - переместить игрока к другому игроку.
  • /v - перевести себя в невидимый режим (обязательно для всех администраторов).

Правила для администраторов

  • Запрещено невыполнение минимальной нормы онлайна (1 час в сутки).
    Наказание: Устный выговор / Строгий выговор / Снятие.
  • Запрещено снимать/заменять наказания, выданные другим администратором (без разрешения старшей администрации).
    Наказание: Устный выговор.
  • Наказания выдаются строго по регламенту.
    Наказание: Беседа / Устный выговор / Строгий выговор.
  • Запрещено выдавать наказания по доказательствам, которые предоставили в ЛС и т.д.
    Наказание: Устный выговор / Строгий выговор.
  • Запрещен багоюз/продажа валюты/услуг и т.д.
    Наказание: Снятие.
  • Администратор обязан хранить доказательства нарушений в течение 48 часов.
    Наказание: Устный выговор / Строгий выговор.
  • Запрещено показывать своё преобладание над игроками.
    Наказание: Строгий выговор / Снятие.
  • Запрещено спавнить технику в людных местах и перед игроками, не имея на то причины.
    Наказание: Устный выговор / Строгий выговор / Снятие.
  • Запрещено вмешиваться в RP процесс.
    Наказание: Устный выговор / Строгий выговор / Снятие.
  • Администратор обязан выдавать наказания беспристрастно.
    Наказание: Устный выговор / Строгий выговор.
  • Администратор обязан знать все правила сервера.
    Наказание: Беседа / Устный выговор / Строгий выговор / Снятие.
  • Администратору запрещено разглашать внутреннюю информацию проекта и работы администрации (из дискорда/форума/ЛС и т.п.).
    Наказание: Снятие.
  • Запрещено угрожать игрокам.
    Наказание: Снятие.
  • Запрещено выдавать игроку hp, оружие, спавн машин и т.д., не имея на то причину.
    Наказание: Строгий выговор / Снятие.

Документация для ивентеров

Ивентеры сервера отвечают за организацию и проведение различных событий и мероприятий для игроков. В их обязанности входит:

  • Создание и разработка концепций для новых ивентов.
  • Подготовка площадок и ресурсов для проведения ивентов.
  • Анонсирование ивентов и привлечение игроков к участию.
  • Проведение ивентов и контроль за их ходом.
  • Оценка и анализ проведенных ивентов, сбор отзывов от игроков.
  • Сотрудничество с другими членами команды для улучшения качества ивентов.

Правила для ивентеров

  • Запрещено невыполнение минимальной нормы онлайна (1 час в сутки).
    Наказание: Устный выговор / Строгий выговор / Снятие.
  • Каждый ивентер обязан знать правила сервера на базовом уровне.
    Наказание: Беседа / Устный выговор / Строгий выговор / Снятие.
  • Запрещено пользоваться правами ивентера для своего блага.
    Наказание: Строгий выговор / Снятие.
  • При создании ивента он должен быть полностью расписан в дискорде.
    Наказание: Устный выговор / Строгий выговор.
  • Запрещено делать ивенты без разрешения ЗГИ+.
    Наказание: Устный выговор / Строгий выговор / Снятие.
  • Запрещено угрожать игрокам.
    Наказание: Снятие.
  • Запрещено выдавать игроку hp, оружие, спавн машин и т.д., не имея на то причину.
    Наказание: Строгий выговор / Снятие.
  • Запрещено разглашать внутреннюю информацию проекта и работы администрации (из дискорда/форума/ЛС и т.п.).
    Наказание: Снятие.

Полезные команды для ивентеров

  • /v - Войти в режим ваниша.
  • /kick [игрок] [причина] - Кикнуть игрока с сервера с указанием причины.
  • /warn [игрок] [причина] - Выдать предупреждение игроку с указанием причины.
  • /set - Команда для установки различных параметров (нужно уточнение).
  • /mute [игрок] [время] [причина] - Выдать игроку мут на определенное время.
  • /ban [игрок] [время] [причина] - Забанить игрока на определенное время с указанием причины.
  • /pardon [игрок] - Помиловать игрока.
  • /invsee [игрок] - Просмотр инвентаря игрока.

Документация для DS-модераторов

DS-модераторы сервера отвечают за поддержание порядка и управление сообществом на Discord-сервере. В их обязанности входит:

  • Модерация чата и форума, обеспечение соблюдения правил.
  • Рассмотрение жалоб от участников сообщества и разрешение конфликтных ситуаций.
  • Выдача предупреждений и наказаний за нарушения правил.
  • Организация и проведение мероприятий на Discord-сервере.
  • Сотрудничество с администрацией и другими модераторами для улучшения сообщества.

Полезные команды для DS-модераторов

Полезные команды для работы с JuniperBot#6999:

  • /kick [пользователь] [причина] - Кикнуть пользователя с сервера с указанием причины.
  • /ban [пользователь] [время] [причина] - Забанить пользователя на определенное время с указанием причины.
  • /unban [пользователь] - Разбанить пользователя.
  • /mute [пользователь] [время] [причина] - Выдать пользователю мут на определенное время.
  • /unmute [пользователь] - Снять мут с пользователя.
  • /warn [пользователь] [причина] - Выдать предупреждение пользователю с указанием причины.
  • /slowmode [время] - Включить режим медленного чата на определенное время.
